# Typescript Template

타입스크립트 개발을 시작하기 위한 기본 템플릿을 제공합니다.

아래의 구성을 포함하고 있습니다.

## 구성

- webpack
- typescript
- jest
- json-server
- eslint
- prettier
- bootstrap
- fortawesome
- axios

## 참고사항

## 설치

- node v14
- npm v6

### NODE 설치

```
$ sudo brew install node

$ sudo npm install -g n

$ sudo n 14.8.0 혹은 sudo n latest
```

### NPM 설치

```
$ sudo npm install -g npm

$ npm install -g npm-check-updates
```

### 패키지 설치

```
$ npm install
```

### 패키지 업데이트

```
$ ncu -u

$ npm install
```

## 시작

### 구동

```
npm run start
npm run db
```

### 빌드

```
// 운영
npm run build

// 개발
npm run dev
```
